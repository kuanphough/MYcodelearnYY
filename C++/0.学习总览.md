**状态**: ✅ 已完成

#### 核心内容

- [x] 枚举
- [x] 构造函数与析构函数
- [x] 继承
- [x] 虚函数与多态
- [x] 纯虚函数与接口
- [x] 访问修饰符

**对应笔记**: `3.ObjectOriented.md`

**关键概念**：
- `class` vs `struct`：默认访问权限不同
- `static`：类级别共享
- `enum` vs `enum class`：作用域枚举
- 继承：`class Derived : public Base`
- `virtual`：虚函数实现运行时多态
- `= 0`：纯虚函数定义接口
- `override`：显式标记重写

**代码示例**：
```cpp
// 虚函数与多态
class Entity {
public:
    virtual std::string GetName() { return "Entity"; }
};

class Player : public Entity {
public:
    std::string GetName() override { return "Player"; }
};

// 纯虚函数接口
class Printable {
public:
    virtual std::string GetClassName() = 0;  // 必须在派生类实现
};
```

**底层原理**：
- 虚函数表 (V Table)：实现动态分派
- 对象内存布局：虚表指针 + 成员变量
- 继承内存布局：基类子对象 + 派生类成员

### 第四阶段：数组与字符串 (ArraysAndStrings)
**状态**: ✅ 已完成

#### 核心内容
- [x] 原生数组
- [x] std::array
- [x] 字符串 (char* 与 std::string)
- [x] 指针算术运算
- [x] const 关键字深度应用
- [x] mutable 与 constexpr

**对应笔记**: `4.ArraysAndStrings.md`

**关键概念**：
- 数组内存连续性
- 数组越界风险
- `std::array`：封装原生数组，提供size()等方法
- 字符串字面量：`const char*` + `\0` 终止符
- `std::string`：动态字符串，支持拼接、查找等操作
- const 三种用法：
  - `const int* a`：指向的内容不可改
  - `int* const a`：指针本身不可改
  - `const int* const a`：两者都不可改
- `mutable`：const 方法中可修改的成员变量
- `constexpr`：编译时常量表达式

**代码示例**：
```cpp
// 原生数组
int example[5];
example[0] = 2;

// 指针算术
int* ptr = example;
*(ptr + 2) = 6;  // 等价于 example[2] = 6

// std::string
std::string name = "Cherno";
name += "hello";
bool contains = name.find("no") != std::string::npos;
```

## 💡 学习心得

### 重要经验
1. **环境很重要**：从Visual Studio迁移到VSCode，需要重新配置环境
2. **多文件调试**：暂时使用 `#include` 大法，后续需要掌握CMake
3. **工具辅助**：Language Reactor 插件大大提高了学习效率
4. **底层理解**：理解内存布局和虚函数表有助于写出更好的代码
5. **AI 辅助学习**：使用 CodeBuddy AI 助手可以更高效地整理笔记和解决问题
6. **定期复习**：复习指针和 const 等概念，加深理解
7. **课程对照学习**：The Cherno 课程与查漏补缺内容高度匹配，可以按照课程进度系统学习
8. **笔记参考**：Jesse Zhang 的学习笔记提供了详细的课程要点，可作为学习参考
