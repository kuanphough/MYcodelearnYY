**对应笔记**: `3.ObjectOriented.md`

**关键概念**：
- `class` vs `struct`：默认访问权限不同
- `static`：类级别共享
- `enum` vs `enum class`：作用域枚举
- 继承：`class Derived : public Base`
- `virtual`：虚函数实现运行时多态
- `= 0`：纯虚函数定义接口
- `override`：显式标记重写

**代码示例**：
```cpp
// 虚函数与多态
class Entity {
public:
    virtual std::string GetName() { return "Entity"; }
};

class Player : public Entity {
public:
    std::string GetName() override { return "Player"; }
};

// 纯虚函数接口
class Printable {
public:
    virtual std::string GetClassName() = 0;  // 必须在派生类实现
};
```

**底层原理**：
- 虚函数表 (V Table)：实现动态分派
- 对象内存布局：虚表指针 + 成员变量
- 继承内存布局：基类子对象 + 派生类成员

### 第四阶段：标准容器 (StringVectorArray)
**状态**: ✅ 已完成

#### 核心内容
- [x] 原生数组
- [x] std::array
- [x] 字符串 (char* 与 std::string)
- [x] 指针算术运算
- [x] const 关键字深度应用
- [x] mutable 与 constexpr

**对应笔记**: `4.StringVectorArray.md`

**关键概念**：
- 数组内存连续性
- 数组越界风险
- `std::array`：封装原生数组，提供size()等方法
- 字符串字面量：`const char*` + `\0` 终止符
- `std::string`：动态字符串，支持拼接、查找等操作
- const 三种用法：
  - `const int* a`：指向的内容不可改
  - `int* const a`：指针本身不可改
  - `const int* const a`：两者都不可改
- `mutable`：const 方法中可修改的成员变量
- `constexpr`：编译时常量表达式

**代码示例**：
```cpp
// 原生数组
int example[5];
example[0] = 2;

// 指针算术
int* ptr = example;
*(ptr + 2) = 6;  // 等价于 example[2] = 6

// std::string
std::string name = "Cherno";
name += "hello";
bool contains = name.find("no") != std::string::npos;
```

### 阶段五：STL与智能指针（推荐立即开始）🔴 高优先级

#### 智能指针 (Smart Pointers)
- `std::unique_ptr` - 独占所有权，自动释放
- `std::shared_ptr` - 共享所有权，引用计数
- `std::weak_ptr` - 弱引用，解决循环引用
- RAII（资源获取即初始化）原则
- 自定义删除器

#### STL 容器
- `std::vector` - 动态数组，最常用容器
- `std::list` - 双向链表，支持快速插入删除
- `std::map` / `std::unordered_map` - 键值对映射
- `std::set` / `std::unordered_set` - 有序/无序集合
- `std::deque` - 双端队列
- `std::stack` / `std::queue` - 适配器容器
